# Wardrobe Clean v16

功能（v16）
- Vision：上傳衣物照 → AI 視覺分析（名稱/分類/材質/厚度/溫度/色彩）
- AI Stylist：依場合+風格 → 自動從指定地點衣櫥挑一套 → 自動收藏
- Outfit DB：LocalStorage DB（可匯入/匯出/重置）
- Health Check：/api/health
- Models Debug：/api/models
- Version：/api/version（no-store）

Vercel 環境變數
- GEMINI_API_KEY = 你的 Google API key
- APP_VERSION = v16.0（可選）
