👗 Wardrobe AI (智能衣櫥日記)
這是一個由 React 打造的 AI 智能衣櫥 PWA (Progressive Web App)，專為手機使用者設計。
🚀 如何在手機上將此專案變成 App (使用 Vercel)
由於您沒有電腦，我們使用最簡單的「雲端部署」方式。
步驟 1：確認檔案
請確保您已在 GitHub 建立了以下檔案：
 * package.json
 * public/index.html
 * src/index.js
 * src/App.js
步驟 2：註冊 Vercel (免費)
 * 用手機瀏覽器打開 Vercel.com。
 * 點擊 "Sign Up"，選擇 "Continue with GitHub"。
 * 授權 Vercel 存取您的 GitHub 帳號。
步驟 3：匯入專案
 * 登入 Vercel 後，點擊 "Add New..." -> "Project"。
 * 在列表中找到 wardrobe-ai，點擊 "Import"。
 * Framework Preset 選擇 Create React App (通常會自動偵測)。
 * 點擊 "Deploy"。
步驟 4：等待部署
Vercel 會自動幫您安裝程式、打包網站。等待約 1-2 分鐘，看到滿版煙火動畫表示成功！
步驟 5：加入主畫面 (變身 App)
 * 點擊 Vercel 產生的網址 (例如 wardrobe-ai.vercel.app)。
 * 在 iPhone Safari 瀏覽器下方點擊「分享按鈕 (方塊箭頭)」。
 * 往下滑，選擇 「加入主畫面 (Add to Home Screen)」。
 * 設定名稱為「衣櫥日記」，點擊「新增」。
🎉 恭喜！現在您的手機桌面上已經有一個真正的衣櫥 AI App 了！
